**有些特征，误以为DL可以学习到，但是其实网络没办法学习出来，所以如果加一些人为引导会提升效果。**

**例如：Deformable convolution, SEnet, Group convolution都是引入人为思考模式加以限制成功提升效果的案例。**

**但也有的特征，DL自身可以很好的学习出来，如果再加人为干预引导，就会画蛇添足，限制了参数优化的自由度。**
